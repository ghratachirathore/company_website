# FlowRem Company Website

## Objective

The objective of this project is to design and develop a responsive multi-page company website for FlowRem. The website showcases the company's services, mission, and contact information while providing a clean and user-friendly experience across different devices.

## Tech Stack

* HTML5
* CSS3
* JavaScript

## Features Implemented

### Home Page

* Hero section
* Call-to-action buttons
* Features section
* Responsive navigation bar
* Footer section

### About Page

* Company introduction
* Mission section
* Team section

### Services Page

* Six service cards
* Process section
* Call-to-action section

### Contact Page

* Contact information section
* Contact form
* Form validation using JavaScript

### Responsive Design

* Mobile-friendly layout
* Flexbox implementation
* CSS Grid implementation
* Responsive navigation menu

## Setup Instructions

1. Clone the repository.

```bash
git clone https://github.com/ghratachirathore/company_website.git
```

2. Open the project folder.

3. Open `index.html` using Live Server.

4. Navigate through all pages and test responsiveness.
